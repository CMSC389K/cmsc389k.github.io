Welcome to CMSC389K!

We are Sashank and Ishaan, your TAs for this semester. We have spent a lot of time designing the coursework for this class, and we hope you will enjoy it! Let us go over a few things:

Course Website

This is the course website. We will be posting announcements on both Piazza and this website. The website does not have a lot of information, but does link to the GitHub repo, which is extremely important. This repo contains all lecture notes, projects, etc.

Syllabus

You can find that in the GitHub repo here.

Class Structure

Before every class, you should read the pre_class.md file for that week, and do whatever it says.

This could involve reading through external links, following a tutorial, or something else. These make sure you are prepared for class that week and should not take long. During class, we will almost always be doing a coding exercise together. This exercise works with the assumption that you did the material in the pre_class.md file.

We will not be posting the complete in-class coding exercises until after class, and there will be little explanation in that file on how it was created (this is how we incentivize coming to class).

Some classes will conclude with a worksheet that you can take home. These will not be graded for accuracy or completion; these are solely for your understanding.

All files from class can be found in the lectures/ folder of the GH repo

Projects

This class is driven by project-based learning. We will be uploading each project the day it is assigned (see syllabus). The requirements are clear in all project descriptions; we are not trying to get you with test cases.

Projects are due at 11:59 PM on the indicated Thursdays before class.

Office Hours and Help

While we have not figured out exactly what time we want to do office hours, feel free to email us (emails provided in syllabus) if you have any questions. We want to help you succeed, so please do not be afraid to reach out!

As soon as we know when we are holding office hours, info will be posted under contact on this site.

We look forward to meeting all of you in person! Have a great rest of break.
